SmartBudget — це веб-застосунок для ведення обліку особистих фінансів, який дозволяє користувачу додавати доходи та витрати, переглядати список транзакцій і обчислювати баланс. 
Застосунок розроблено на Python з використанням Flask для бекенду та SQLite для зберігання даних, а інтерфейс реалізовано за допомогою HTML та CSS. В структурі проекту є папки templates для HTML-шаблонів, static для CSS та інших статичних файлів, файл app.py для запуску застосунку та requirements.txt з переліком залежностей.
Щоб запустити застосунок, потрібно встановити залежності через pip install -r requirements.txt і виконати python app.py, після чого відкрити адресу http://127.0.0.1:5000/ у браузері.
